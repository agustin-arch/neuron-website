### 2. EL PROBLEMA

**Título:**
```
El costo real de la cobranza tradicional
```

**Pain Points (4):**

**1. Costos operativos altos**
```
Equipos de cobranza requieren contratación constante, capacitación continua y tienen alta rotación. Los costos operativos crecen linealmente con el volumen, limitando tu capacidad de escalar y mejorar márgenes.
```

**2. Recovery rates bajos**
```
A mayor volumen, menor calidad. La efectividad se deteriora cuando los equipos están saturados.
```

**3. Compliance complejo**
```
Con humanos no garantizas 100% de cumplimiento. Con Neuron, cada llamada cumple automáticamente.
```

**4. Falta de data accionable**
```
Sin visibilidad clara de qué funciona, optimizar tu operación es imposible.
```

**Layout sugerido:**
- Grid horizontal (4 columnas, 1x4)
- Todo visible en una pantalla (100vh)
- Sin íconos o con íconos SVG minimalistas

### 5. CÓMO FUNCIONA

**Título:**
```
Cómo funciona
```

**Proceso (4 fases):**

**Fase 1: Discovery**
```
Entendemos tu operación actual: flujos de cobranza, integraciones existentes, pain points específicos y objetivos de negocio. Mapeamos el journey completo de tus clientes.
```

**Fase 2: Configuración**
```
Personalizamos los agentes con el tono y estrategia de tu marca. Integramos con tu CRM, sistemas de pago y bases de datos. Entrenamos los modelos con tu contexto empresarial.
```

**Fase 3: Piloto**
```
Deploy limitado con un subconjunto de tu cartera. Monitoreo en tiempo real, ajustes continuos basados en performance real. Validación de efectividad antes del rollout completo.
```

**Fase 4: Escala**
```
Rollout completo a toda tu operación. Optimización continua basada en data. Soporte dedicado y evolución constante del sistema. Podemos llegar a este punto en tan solo 4 semanas.
```

**Layout sugerido:**
- Timeline horizontal (4 columnas)
- Visual de progreso/flujo entre fases
